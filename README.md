# Projet Deep Learning
Ce projet compare deux architectures de Deep Learning, les CNN et Vision Transformers (ViT), pour la classification d’images. Des datasets sont créés ou modifiés afin d’analyser leurs forces et limites sur textures, relations globales et images dégradées. Les modèles sont entraînés avec PyTorch/TensorFlow et évalués via métriques et visualisations.
